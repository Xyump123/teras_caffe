# ☕ Teras Caffe

Aplikasi pemesanan menu berbasis web menggunakan CodeIgniter 4.

## ✨ Fitur

- Login & Register Admin
- Manajemen Menu
- Transaksi Pemesanan
- Cetak Struk
- Dashboard Admin

## 🛠️ Teknologi

- PHP (CodeIgniter 4)
- MySQL
- Bootstrap / CSS

## 🚀 Cara Menjalankan

1. Clone repo
2. Import database `teras_caffe.sql`
3. Jalankan di XAMPP
4. Akses di browser: http://localhost/teras_caffe
